Caso de Prueba 3: Flujo de Error – Archivo no compatible
Atributo	Detalle
ID	TC-YT-003
Nombre	Rechazo de archivo no compatible
Tipo	Funcional – Flujo de error
Precondiciones	Usuario autenticado
Datos de prueba	Archivo: audio_podcast.wav (formato de audio no soportado como video)
Pasos	
1. Iniciar sesión
2. Hacer clic en “Subir video”
3. Seleccionar archivo audio_podcast.wav
Resultado esperado	Mensaje de error: “Formato de archivo no compatible. Intenta con un archivo .mp4, .mov, .avi, etc.”
Evidencia	Captura del mensaje de error en la interfaz de subida
