
 Examen de Casos de Prueba – QA Functional Testing

 Descripción del Proyecto
Este repositorio contiene el desarrollo del examen práctico de QA, enfocado en el diseño, documentación y ejecución de **casos de prueba** sobre flujos funcionales, alternos y de error. Los escenarios están basados en plataformas reales como **Saga Ripley**, **YouTube** y **Facebook**.

 Objetivo
Evaluar la capacidad para:
- Diseñar casos de prueba efectivos
- Documentar pasos, datos y resultados esperados
- Evidenciar la ejecución mediante capturas, reportes y logs
